# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by emailing [uzairwaseem29@gmail.com](mailto:uzairwaseem29@gmail.com) with:

- the affected URL or file;
- the steps needed to reproduce the issue;
- the expected and observed behaviour; and
- any proof-of-concept material that can be shared safely.

Do not include secrets or personal data in a public GitHub issue. I will acknowledge a useful report, assess its impact, and coordinate remediation before public disclosure.

## Scope

This repository powers the public portfolio at [uzairwaseem.com](https://uzairwaseem.com). The linked product repositories have their own security boundaries and documentation. Public recruiter demos must use synthetic data and must not expose private operational records.
